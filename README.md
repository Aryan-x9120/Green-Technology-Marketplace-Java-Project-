# Green Tech Marketplace 

**Green Tech Marketplace** is a Java Web Application designed to promote sustainable living by connecting vendors of eco-friendly products with conscious consumers. The project is built using standard **Java EE** technologies, following the **MVC (Model-View-Controller)** architecture and **Maven** for build management.

##  Project Overview

This application allows three types of users to interact with the system:
*   **Admin:** Manages users and oversees platform activity.
*   **Vendor:** Adds and manages eco-friendly products.
*   **Customer:** Browses products and views details.

##  Tech Stack

*   **Language:** Java (JDK 1.8+)
*   **Web Technologies:** Servlets, JSP, JSTL, HTML5, CSS3
*   **Build Tool:** Apache Maven
*   **Database:** MySQL
*   **Server:** Apache Tomcat (v9.0 recommended)
*   **Concepts Implemented:** MVC Pattern, DAO Pattern, JDBC, Multithreading (Email), Exception Handling.

---

##  Key Features

1.  **User Authentication:** Secure Login and Registration with role-based redirection (Admin/Vendor/Customer).
2.  **Product Management:** Vendors can add, update, and delete products.
3.  **Dashboard Views:** dynamic dashboards customized for every user role.
4.  **Database Connectivity:** Robust MySQL integration using JDBC and Singleton/Factory patterns.
5.  **Asynchronous Operations:** Multithreading implementation for sending email notifications without freezing the UI.
