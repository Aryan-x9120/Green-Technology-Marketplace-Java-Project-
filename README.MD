# Green Technology Marketplace

## Project Overview
A Java-based web application connecting eco-friendly vendors with sustainable-conscious customers.

## Technologies Used
- **Backend:** Java Servlets, JSP, JDBC
- **Database:** MySQL
- **Frontend:** HTML, CSS, JavaScript
- **Architecture:** MVC Pattern with Service Layer

## Features
- User Authentication (Admin, Vendor, Customer)
- Product Management with Image Support
- Inventory Synchronization (Thread-safe)
- Admin Dashboard
